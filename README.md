## To-Do List App in Java

A simple command-line To-Do List application written in Java. Users can add tasks, mark them as completed, and view the list. Tasks are also saved to a file (todolist.txt) so that you can keep a record.

### Features
Add tasks to a to-do list
Mark tasks as completed
Display tasks with a [x] for completed and [ ] for pending
Save tasks to a text file
Simple command-line interface
